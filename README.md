# About Blank Page Maker
## Release 1.0
## Second latest version: [None](about:blank)
-------------------------------------------------
# Features
## Feature 1
### Tag your texts with ID's!
### Example
```aboutblankpagemaker
"Hello world!"[id=1]
```
## Feature 2
### Move your texts from texts putted in ID's!
### Example
```aboutblankpagemaker
"Hello world!"[id=1]
"I made an about:blank page!"[id=2, down[id=1]]
"Hi again!"[id=3, right[id=1]]
```
## Feature 3
### Make your texts colored!
### Example
```aboutblankpagemaker
"Hello "[id=1]
"world!"[id=2, right[id=1], color=0,255,0
```
## Feature 4
### Color your background!
### Example
```aboutblankpagemaker
BGcolor=255,255,255
```
